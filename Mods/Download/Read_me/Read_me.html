Where should I install it?
Walls (.wll) The Sims/GameData/Walls
Floors (.flr) The Sims/GameData/Floors
Objects (.iff) The Sims/Downloads or The Sims/Downloads/Around the Sims
Skins (.bmp, .cmx, .skn) The Sims/GameData/Skins
If you're a PC user, have a look on your Skins folder, to see if you don't have already the mesh you downloaded with the skins, because I had to rename them to make them "Mac compatible". (You can probably use it without any risk, but it's stupid to have twice the same mesh, isn't it? =))
