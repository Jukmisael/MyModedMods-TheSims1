KGB Archiver 1.2 (Windows GUI)

KGB Archiver es la herramienta de archivado con un increíble promedio de compresión. Sobrepasa con creces a herramientas como 7zip o UHARC en términos de habilidad. Desafortunadamente su alto poder de compresión requiere un alto poder de Hardware (Recomendamos un procesador de 1.5Ghz y 256Mb Ram como mínimo). Una de las ventajas de KGB Archiver es su algoritmo de encriptación AES-256 él que se utiliza para los archivos. Una de las mejores encriptaciones conocidas por el ser humano.

Requerimientos de Hardware
Mínimo  
Procesador 1,5 GHz (32-bit)
RAM 256 MB
Espacio Libre en Disco Duro 4 MB
Sistema Operativo Windows NT/2000/XP/2003
Recomendado  
Procesador 3,0 GHz (64-bit)
RAM 1,5 GB
Espacio Libre en Disco Duro 4 MB
Sistema Operativo Windows XP x64 SP1/2003 x64 SP1

Línea de Comandos
Compresión  
kgb_arch_compress.exe "archivo1" "archivo2" "archivo3" en lugar de archivo1, archivo2, archivo3,... puedes utilizar las rutas del directorio con backslash (\) al final
Extracción  
kgb_arch_decompress.exe /s "archivo.kgb" "ruta" extrae archivos kgb/kge/sfx al directorio "ruta" (argumento óptimo), /s - silent mode (argumento óptimo)
